# AI-First CRM System

A full-stack AI-powered CRM application built using **React, FastAPI (Python), MySQL, and Groq LLM API**.  
This project demonstrates modern CRM features with AI integration for smart user assistance and customer management.

---

## 🚀 Tech Stack

### Frontend
- React.js
- HTML, CSS, JavaScript
- Axios

### Backend
- Python (FastAPI)
- SQLAlchemy
- Uvicorn

### Database
- MySQL

### AI Integration
- Groq API (LLM)

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 👤 Customer Management (Create, Read, Update, Delete)
- 🤖 AI Chatbot using Groq LLM API
- 📊 CRM Dashboard for data visualization
- 🗄️ MySQL Database integration
- ⚡ Fast and scalable FastAPI backend
- 🎯 Clean and responsive React UI
